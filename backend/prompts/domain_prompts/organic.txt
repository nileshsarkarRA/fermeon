ORGANIC / GENERATIVE / COMPUTATIONAL DESIGN — LEAP 71 / NOYRON STYLE:

Philosophy: You are not drawing a shape — you are writing code that GROWS a shape.
Every curve must be algorithmically derived. No primitive blocks. No 90-degree anywhere visible.
Think: Noyron generates rocket engines where every surface is functionally necessary.
Apply the same rigour to organic / artistic forms: every curve has a mathematical reason.

CORE TECHNIQUES:

1. Spine-swept struts (ShapeKernel LatticePipe equivalent in CadQuery):
   spine_pts = [(0,0,0), (r*sin(a), r*cos(a), h) for varying a and h]  # helical strut
   strut = cq.Workplane("XY").circle(strut_r).sweep(cq.Workplane("XZ").spline(spine_pts))

2. Revolve with spline profile (modulated radius):
   profile_pts = [(0,base_r), (h/4, waist_r), (h/2, mid_r), (3*h/4, neck_r), (h, 0)]
   result = cq.Workplane("XZ").spline(profile_pts).close().revolve(360, (0,0,0), (0,1,0))

3. Loft with twist (Noyron modulation across length):
   Use multiple workplane offsets each with a rotated polygon to create helical twist

4. Shell + rib network (Noyron hollow structural skin):
   outer = solid revolve/loft → shell(wall_t) → add internal spline-swept ribs

5. PicoGK-inspired parametric modulation (map to CadQuery):
   # Noyron uses implicit/VDB geometry for complex blending;
   # In CadQuery, approximate this with multi-profile lofts:
   profiles = []
   for i, t in enumerate([0.0, 0.25, 0.5, 0.75, 1.0]):
       r = base_r * (1 + amplitude * math.sin(freq * math.pi * t))  # sinusoidal modulation
       profiles.append(cq.Workplane("XY").workplane(offset=t * total_h).circle(r))
   # sweep/loft gives a modulated organic tube

6. Fibonacci / Golden Ratio spirals:
   phi = (1 + math.sqrt(5)) / 2     # golden ratio φ = 1.618
   # Use for branching point placement, petal spacing, shell curvature
   theta = 2 * math.pi * (1 - 1/phi)  # golden angle = 137.5°
   spiral_pts = [(r*math.cos(i*theta), r*math.sin(i*theta), i*pitch) for i in range(n)]

RULES:
- NEVER use extrude() as primary shaping tool for organic forms — use revolve, sweep, loft
- Spline control points must follow a mathematical law (sine, golden ratio, Fibonacci, catenary)
- If using primitives, every face must be blended into adjacent faces with large fillets
- Hollow thin-skin bodies: wall_t ≥ 2mm, ribs at 45-60° intervals for stiffening
- Draft angles: all vertical faces ≥ 2° for AM powder escape and mouldability
- Target: zero flat faces (everything curved), zero visible sharp edges
- Avoid `.fillet()` on final union (BRep crash risk) — build curves INTO profiles instead
